Que es Django?

Django es un framework de desarrollo web de alto nivel y código abierto, escrito en Python, que permite crear aplicaciones web de forma rápida, limpia y eficiente. Fue desarrollado originalmente por un equipo de programadores de un periódico en Kansas, Estados Unidos (el Lawrence Journal-World), y liberado como software libre en 2005. Desde entonces, ha crecido hasta convertirse en uno de los frameworks más populares del ecosistema Python. Su objetivo principal es automatizar tanto como sea posible y seguir el principio DRY (Don't Repeat Yourself, es decir, "no te repitas") para evitar redundancias en el código, mejorar la mantenibilidad y acelerar el desarrollo.

Función

Django automatiza y organiza todas las tareas necesarias para construir un sitio o aplicación web, como: 1.Gestionar la lógica del servidor Por ejemplo: ¿qué pasa cuando un usuario entra a una página?, ¿qué datos se muestran?, ¿cómo se procesan formularios?, ¿cómo se guardan los datos? 2.Conectarse y manejar bases de datos Django convierte clases Python en tablas de base de datos y te permite guardar, consultar, actualizar y eliminar datos sin escribir SQL. 3.Definir y controlar las rutas de navegación (URLs) Establece qué vista se debe ejecutar cuando un usuario entra a cierta dirección web. Renderizar contenido dinámico en páginas HTML Permite insertar datos (como listas de productos, perfiles, noticias) en plantillas HTML usando su motor de plantillas. 4.Controlar autenticación y permisos Incluye herramientas listas para usar para crear usuarios, proteger páginas, iniciar sesión, cerrar sesión, y manejar permisos. 5.Generar interfaces administrativas automáticamente Ahorra tiempo creando un panel completo para que los administradores gestionen la base de datos sin necesidad de codificarlo desde cero. 6.Seguridad web integrada Django protege automáticamente de muchas amenazas comunes como inyecciones SQL, XSS o CSRF.
